What is pytorch ?

It’s a Python-based scientific computing package targeted at two sets of audiences:

  • A replacement for NumPy to use the power of GPUs
  • a deep learning research platform that provide maximum flexibility and speed

1 Tensors

Tensors are similar to NumPy’s ndarrays,with the addition being that Tensors can also be used on a GPU to accelerate computing.

  • construct a 5x3 matrix, uninitialized: x = torch.empty(5, 3)
  • construct a randomly initialized matrix: x = torch.rand(5, 3)
  • construct a tensor directly from data: x = torch.tensor([5.5, 3])
  • create a tensor based on an existing tensor. These methods will reuse properties of the input tensor,e.g. dtype,unless new values are provided by user.: x = x.new_ones(5, 3, dtype=torch.double)

    1.1 Get its size:

    x.size() # out: torch.Size([5, 3]). 是一个类。

NOTE: torch.Size is in fact a tuple. so it supports all tuple operations.

1.2 Operations

  • result = x + y

1.6 package中init.py的作用

  • Python中package的标识，不能删除
  • 定义”all“来模糊导入
  • 编写Python代码，起到像模块的作用。例如torch包中的”init.pyi”就写了很多函数和类。但不建议在init中写Python模块，可以在包中创建另外的模块来写，尽量保证init.py的简单。

    2 AUTOGRAD: AUTOMATIC DIFFERENTIATION

    The autograd package provide automatic differentiation for all operations on Tensors. It is a define-by-run framework, which means that your backprop is defined by how your code is run, and that every single iteration can be different.

2.1 Tensor

不懂的句子：

  1. To stop a tensor from tracking history, you can call .detach() to detach it from the computation history, and to prevent future computation from being tracked.
  2. Generally speaking, torch.autograd is an engine for computing vector-Jacobian product.

重要的几个属性和方法：

  1. .requires_grad : (True or False)
  2. .backward() : have all the gradients computed automatically.
  3. .grad : The gradient for this tensor will
  4. .detach() : to detach a tensor from the computation history, and to prevent future computation from being tracked
  5. .grad_fn : reference a Function that has crated the Tensor
